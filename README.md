# Wi-Fi Profile Cleanup

A PowerShell script to remove old or unnecessary Wi-Fi profiles from the system.

## Features

- Lists saved Wi-Fi profiles
- Removes stale or unwanted profiles
- Helps improve security and network performance

## Usage

Run PowerShell as administrator and execute:

.\WifiCleanUp.ps1

## Notes

- Ensure you do not remove active or important profiles.
- Useful for shared or mobile devices that connect to multiple networks.

## Disclaimer

Use responsibly. Removing profiles will require re-entering credentials to reconnect.
